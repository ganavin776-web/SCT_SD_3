#  SkillCraft Technology Internship

## 📌 Task 3 – 🧩 Sudoku Solver (GUI) 

This project is a GUI-based Sudoku Solver built using Python and Tkinter.

## 🚀 Features
- Interactive Sudoku grid
- Difficulty levels (Easy, Medium, Hard)
- Automatic puzzle generation
- Solves puzzles using Backtracking Algorithm
- Clean and user-friendly interface

## 🛠️ Technologies Used
- Python
- Tkinter (GUI)
- Backtracking Algorithm

## ▶️ How to Run
1. Download or clone the repository
2. Run the Python file:
   python sudoku_solver.py

## 🎯 Outcome
This project demonstrates problem-solving using algorithms and building interactive GUI applications.
